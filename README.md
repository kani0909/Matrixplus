# s21_matrix+ (Matrix Library in C++)

Реализация библиотеки для работы с матрицами на C++ с использованием объектно-ориентированного подхода.

## О проекте

Библиотека предоставляет класс `S21Matrix` для выполнения основных матричных операций: сложение, вычитание, умножение, транспонирование, вычисление определителя, матрицы алгебраических дополнений и обратной матрицы. Реализованы конструкторы копирования и перемещения, деструктор, операторы доступа по индексам и перегрузка арифметических операторов.

## Технологии и инструменты

- **C++20** (стандарт языка)
- **Google Test** (модульное тестирование)
- **Makefile** (сборка)
- **Git** (контроль версий)

## Реализованные операции

| Операция | Метод | Оператор |
|----------|-------|----------|
| Сравнение матриц | `EqMatrix()` | `==` |
| Сложение | `SumMatrix()` | `+`, `+=` |
| Вычитание | `SubMatrix()` | `-`, `-=` |
| Умножение на число | `MulNumber()` | `*` (с числом), `*=` |
| Умножение матриц | `MulMatrix()` | `*` (с матрицей), `*=` |
| Транспонирование | `Transpose()` | — |
| Матрица дополнений | `CalcComplements()` | — |
| Определитель | `Determinant()` | — |
| Обратная матрица | `InverseMatrix()` | — |
| Доступ по индексам | `(i, j)` | `()` |

## Конструкторы и деструктор

- `S21Matrix()` — конструктор по умолчанию
- `S21Matrix(int rows, int cols)` — параметризованный конструктор
- `S21Matrix(const S21Matrix& other)` — конструктор копирования
- `S21Matrix(S21Matrix&& other)` — конструктор перемещения
- `~S21Matrix()` — деструктор
